Site Plan

FitStart Blueprint

Site Name

Name: FitStart Blueprint

Reason: The name highlights the site as a clear starting point for beginners with a structured, step-by-step path to fitness.

Optional domain: fitstartblueprint.com

Site Purpose

FitStart Blueprint provides beginner-friendly fitness education, simple workout planning, and practical tips to help new exercisers build confidence and consistency. Visitors can explore foundational concepts, choose goals, and view example routines that adapt to their needs.

Scenarios

  • Which beginner workout plan should I follow if my goal is general health?
  • How many days per week should I train to start safely?

Color Schema

Deep Blue (#0B3D91) – headings, nav, primary buttons

Warm Amber (#F4B740) – accents, highlights, callouts

Light Gray (#F6F7FB) – page background

Charcoal (#1F2937) – body text

Typography

  • Montserrat – headings and navigation
  • Source Sans 3 – body text and form labels

Wireframes

Mobile View

  1. Logo + navigation drawer
  2. Hero: headline, short intro, call-to-action
  3. Goal selector buttons (strength, fat loss, general health)
  4. Workout preview cards (3)
  5. Tips strip (safety, recovery, consistency)
  6. Newsletter or contact form
  7. Footer with references link

Desktop View

  1. Top bar: logo left, navigation right
  2. Two-column hero: text left, illustration right
  3. Three goal selector cards in a row
  4. Three workout cards in a row
  5. Tips section split into two columns
  6. Form with image beside it
  7. Footer with references link
